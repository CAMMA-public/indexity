Indexity Annotation Platform

Cloud based, collaborative video annotation platform

  • Video storage and streaming
  • Video list searching and filtering
  • Create annotations
  • Visualize annotations
  • Edit and update annotations
  • Collaborative mode allows for real time updates between users
  • Public API allows for automation on video & annotation management
  • Export/Import annotations from a formatted JSON file

PLATFORM Overview

Intuitive User Interface and flexible public API allow to read, write, update and extend the annotations database
by using automated algorithms and a user base

Manage the videos

Access a list of shared videos. Check their stats: See who already annotated it, how many annotations are there, which label are used...

Keep track of the videos status. Set the status of the videos you're going to annotate, you are annotating or you already annotated.

Annotate videos

Draw and label bounding boxes around the objects of interest in your videos and follow them while the video is playing.

Use edit mode to update the labels, positions and sizes or to cut the annotations.

Export/Import annotations in JSON format.
